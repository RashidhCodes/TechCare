# 🛠️ TechCare — Device Repair Booking App

A fully functional Android mobile application built with Java and SQLite 
for booking device repair services.

## 📱 Features
- User Registration and Login with validation
- SQLite database integration
- Browse device repair categories
- Submit repair booking requests
- Real-time repair tracking
- Alert notifications center
- FAQ support center

## 🖥️ Screens
| Screen | Description |
|--------|-------------|
| Splash | App launch screen |
| Login | Secure user authentication |
| Register | New user account creation |
| Home Hub | Dashboard with repair categories |
| Service Detail | Repair info and pricing |
| Request Form | Booking form with date picker |
| Order Summary | Booking confirmation |
| Tracking | Real-time repair status |
| Alert Center | Service notifications |
| Support | FAQ and contact options |

## 🗄️ Database
- SQLite with 2 tables: Users and Bookings
- Full CRUD operations
- Session management with SharedPreferences

## 🛡️ Validation
- Email format validation
- Phone number validation
- Password minimum length
- Duplicate email prevention
- Future date enforcement

## 🔧 Built With
- Java
- Android Studio
- SQLite
- XML Layouts
